# AWS

1. EC2 - elastic compute cloud
2. VPC - virtual private cloud (network in the cloud)
3. S3 - simple storage service - files
4. RDS - databases
5. RD3 - DNS service
6. ELB - elastic load balancing
7. Autoscaling
8. Lambda

Services scope: global (IAM, Route), region (S3), availability zone (EC2)
